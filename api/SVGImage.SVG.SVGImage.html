Class SVGImage

Namespace
SVGImage.SVG

DotNetProjects.SVGImage.dll

This is the SVG image view control. The image control can either load the image from a file SetImage(string) or by setting the Drawing object through SetImage(Drawing), which allows multiple controls to share the same drawing instance.
